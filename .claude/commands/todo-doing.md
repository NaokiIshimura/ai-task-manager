# /todo-doing コマンド

## 概要
todoリストからタスクを選択してdoingに移動し、実行を開始する

## 使用方法
```
/todo-doing <タスク番号>
```

## 処理内容
1. todo.mdのtodoリストから指定された番号のタスクを取得
2. doingセクションに「x. YYYYMMDD_<todo>」形式でタスクを追加
3. todoセクションから該当タスクを削除
4. ./doing配下に「YYYYMMDD_<todo>」フォルダを作成
5. タスクの実行を開始

## 例
```
/todo-doing 1
```

## 動作例
実行前:
```
# todo
1. 今日の東京都の天気を確認して
2. 明日の東京都の天気を確認して

# doing
```

実行後 (/todo-doing 1):
```
# doing
1. 20250809_今日の東京都の天気を確認して

# todo
1. 明日の東京都の天気を確認して
```
